# Note Master 五线谱音符练习

一个面向小学生的五线谱识音与听音练习小游戏。  
支持高音谱号、低音谱号、88 键钢琴、触摸操作、成绩评定和历史记录。

在线访问：<https://notemaster.17hiking.cn>

## 功能亮点

- 支持高音谱号和低音谱号随机出题
- 支持 88 键钢琴，范围为 `A0` 到 `C8`
- 支持鼠标点击、触摸点击、方向键选择
- 出题方式可选：
  - 谱面
  - 声音
  - 谱面 + 声音
- 可选择练习组别范围，例如第 2 组到第 5 组
- 答题模式可选：
  - 按题数练习
  - 按时间练习
- 支持自动下一题或手动下一题
- 支持查看答案
- 答题后正确琴键显示绿色，错误琴键显示红色
- 练习结束后弹出成绩卡，显示答对、答错、正确率和评定
- 自动记录历史练习结果
- 适配 MacBook Air、iMac、iPad、iPad Air，并支持触摸

## 成绩评定

练习结束后会根据正确率给出英文 + 中文评定：

| 正确率 | 评定 |
| --- | --- |
| 100% | Perfect Pitch · 完美小乐手 |
| 90% 及以上 | Excellent · 优秀 |
| 75% 及以上 | Great Job · 表现很棒 |
| 60% 及以上 | Keep Going · 继续加油 |
| 60% 以下 | Try Again · 再试一次 |

## 使用方式

直接打开在线地址即可使用：

<https://notemaster.17hiking.cn>

也可以下载本仓库后，直接用浏览器打开 `index.html`。

## 操作说明

- 点击钢琴键：选择答案
- 触摸钢琴键：选择答案
- 方向键 `←` / `→`：移动当前选择的琴键
- `Enter` 或空格：确认当前选择
- `听当前音`：播放当前题目的音
- `查看答案`：显示正确答案，不计入对错
- `重新开始`：重置当前练习

## 技术说明

这是一个纯前端静态网页，不需要后端服务。

- HTML / CSS / JavaScript
- SVG 绘制五线谱与音符
- Web Audio API 播放音高
- localStorage 保存历史成绩
- GitHub Pages 托管

## 适合人群

适合正在学习五线谱、钢琴基础和音名识别的儿童，也适合老师或家长用来做课堂/课后练习。
